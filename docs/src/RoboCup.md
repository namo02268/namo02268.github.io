# ロボカップジュニア
小学生～高校生の間、[ロボカップジュニアのサッカーリーグ](https://www.robocupjunior.jp/info.html)に参加していました。自律型ロボットによる競技で、コントローラーを用いた操作などは禁止されているため、マイコンやプログラムの知識が必要です。ルールは、2台のロボットで構成される各チームが、相手のゴールにボールを入れて得点を競うというものです。
